# Write

## 基礎文風

- 不使用 emoji
- 不使用 em-dash（「——」或「—」）
- 語氣直接、自然，不過度熱情
- 結尾要自然收尾，不用「這篇很值得讀！」、「快去試試看！」等強推句

## 社群貼文

- 篇幅簡短，重點明確，適合直接貼到群組或社群
- 開頭點出主題，不用套話暖場
- 段落之間保持節奏，不要太密
- 結尾用一句自然的補充或延伸，讓讀者知道為什麼值得關注，但不要用強推語氣

## 不要做的事情

- 不用 emoji
- 不用 em-dash
- 不用強推式結尾（例如「這篇很值得讀！」「快去看看！」）
- 不用過於正式或像新聞稿的語氣
