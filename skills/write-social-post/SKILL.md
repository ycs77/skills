# Write Social Media Post

## 語氣與個性

- 像真實的人在說話，不像在唸稿或寫公告
- 句子長短有變化，保持節奏感，不要每句都一樣長
- 觀點直接說出來，不用迂迴鋪墊或過度解釋
- 情緒可以自然流露，但不刻意煽動或誇張
- **整篇不出現第一人稱**（「我」「我覺得」「我認為」「個人覺得」「筆者」都不寫），判斷直接落下來，觀點歸屬從語氣就看得出來；動詞也用口語的「覺得」而不是「認為本文應當」這種公文體
- 不堆砌形容詞，讓事實和例子說話
- 當語氣想往上揚，帶出輕鬆、開心、或自嘲的情緒時，可以在句尾緊接「~」（開心/愉悅）或「XD」「XDD」（自嘲/好笑），不加空格；這種情緒自然出現在段落收尾，整篇一到兩次就夠，不要刻意安排

## 結構與寫法

- 篇幅簡短，重點明確，適合直接貼到群組或社群
- 開頭點出主題，不用套話暖場
- 開頭第一行是用 **粗體** 包裹的標題
- 貼文內容可以用 **粗體** 標出真正重要的詞或句子，但不要每句都用，保持稀缺感才有效果
- 段落之間保持節奏，不要太密
- **鋪陳順序：先報導，後觀點**。前面段落只鋪事實、做法、數據，不夾「值得關注」「滿有意思」這類主觀詞；最後一段才收束成判斷或視角，讓讀者先拿到資訊、再拿到觀點
- 結尾的觀點要**具體**：講「對誰有感」「哪裡微妙」「什麼情況下要保留」這種有判斷力的話，不要寫模糊的推薦語
- 如果有附上連結（文章、repo 等），獨立放在最後一行，不要夾在段落裡

## 禁止事項

- 不使用 emoji：讓文字本身承重，不靠符號撐氣氛
- 不使用 em-dash（「——」或「—」）：容易讓句子顯得戲劇化或像在翻譯腔
- 不用強推式結尾（例如「這篇很值得讀！」「快去試試看！」「快去看看！」「快去做這件事！」）：讀者自己判斷，不需要被催促去做任何事
- 不用過於正式或像新聞稿的語氣：這是人寫給人看的，不是公告
- 不用**結構宣告詞**開段（例如「核心想法是」「重點是」「主要特點包含」「值得關注的是」）：人不會先預告「我要講重點了」，直接把事實或判斷說出來就好
